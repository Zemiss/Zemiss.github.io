# 机器视觉技术

## 第2章 数字图像的概念与性质

**二值图像、灰度图像、彩色图像基本概念**

- 灰度图像是二维矩阵，每个元素是一个灰度值；
- 彩色图像通常是三通道矩阵；
- 二值图像只有0/1或黑/白两类像素。

**采样、量化的概念**

- 采样是把连续空间坐标离散成像素网格
- 量化是把连续像素值转化为数字等价量
- 采样不足会导致空间细节丢失、图像块状化；
- 量化级数太少会导致灰度层次断裂。

**直方图、积分图**

- 直方图统计每个灰度级出现的像素数量或概率，可以用于阈值分割、直方图均衡化、图像对比度分析。
- 积分图像快速计算矩形区域内像素和。

**距离变换算法**

算法：距离变换/斜切（按照某种距离度量D，对大小为M×N的图像的子集S计算距离变换）

1. 创建大小为M×N的二维数组F，子集S元素位置置为0，其他位置置为无穷大
2. 遍历图像，从上到下，从左至右。利用上方和左侧邻接像素(AL集合)，计算：$F(p)$
3. 遍历图像，从下到上，从右至左。利用下方和右侧邻接像素(BR集合)，计算：$F(p)$
4. 数组F的结果即为子集S的斜切

**卷积概念及计算**

输出图像像素 g(m,n) 是输入图像 f(m,n) 在一个局部邻域内像素亮度的线性组合，邻域中每个像素的贡献由系数 h 加权；这个 h 也叫滤波器、卷积掩模、核、窗口。

公式写成：

$$g(m,n)=\sum_{(k,l)\in \Omega} f(k,l)h(m-k,n-l)$$


## 第3章 图像预处理

**灰度级变换的基本概念**

- 亮度校正与像素位置相关，在改变像素的亮度时要考虑该像素原来的亮度及其在图像中的位置
- 灰度级变换与像素位置无关，在改变像素的亮度时无需考虑其在图像中的位置
- 变换T，将原来在范围$[p0,pk]$内的亮度$p$变换为一个新范围$[q0,qk]$内的亮度

**直方图均衡化算法**

用累计分布函数把灰度重新映射，使灰度分布更均匀，从而增强对比度

输入：$M \times N$ 灰度图像，灰度级总数 $k=256$
1. 初始化长度为 $k$ 的直方图数组 $\boldsymbol{H}$，置全部元素为0；
2. 遍历全图像素，统计各灰度级像素数量，生成原始直方图 $\boldsymbol{H}$；
3. 基于 $\boldsymbol{H}$ 计算累计直方图 $\boldsymbol{H_c}$：$H_c[p]=\sum\limits_{i=0}^p H[i]$；
4. 构造均衡化灰度映射查找表：
$$
T[p] = \frac{k-1}{M \cdot N}\big(H_c[p]-H_c[0]\big),\quad p=0,1,2,\dots,k-1
$$
1. 再次遍历原图所有像素，根据查找表 $T$ 替换每个像素灰度，输出均衡图像。

**几种滤波器（均值、高斯、中值）的作用**

- 均值滤波简单但容易模糊边缘；
- 高斯滤波考虑距离权重，是Canny等算法前常用的去噪步骤；
- 中值滤波对椒盐噪声特别有效，因为它不是线性平均，而是取邻域中位数。

**边缘检测数学表达**

信号一阶微分最大值/两阶微分过零点检测。

**Canny边缘检测基本原理与算法**

- 检测标准：要有好的检测结果，不丢失重要的边缘，不应有虚假的边缘
- 定位标准：实际边缘位置与检测到的边缘位置间的偏差最小
- 单响应标准：对实际上的同一边缘要有低的响应次数

2. 对g中的每个像素计算梯度的大小和方向
3. 根据像素梯度方向，通过非极大值抑制获取边缘位置

## 第4章 二值图像处理

**二值图像形态学：膨胀、腐蚀、开闭运算的计算**

- 开运算：先腐蚀再膨胀 𝐴 ∘ 𝐵 = (𝐴 ⊖ 𝐵) ⊕ B
- 使图像的轮廓变得光滑，断开狭窄的间断，去掉细小的突出物
- 闭运算：先膨胀再腐蚀 𝐴 ∙ 𝐵 = (𝐴 ⊕ 𝐵) ⊖ B
- 使图像的轮廓变得光滑，将狭窄的缺口连接形成细长的弯口，并填充比结构元素小的洞

## 第5章 传统特征检测

**Harris 角点检测**

1. 对于图像上的每一个点(𝑥𝑖, 𝑦𝑖) ，计算自相关矩阵M
2. 对于每一像素点(𝑥𝑖, 𝑦𝑖)计算角点响应函数𝑅(𝑥𝑖, 𝑦)
3. 设定阈值T，将角点响应函数𝑅(𝑥𝑖, 𝑦𝑖)中所有低于T的值设为0
4. 在窗口范围𝑟内进行非极大值抑制：遍历角点响应函数，若某个像素的灰度值在窗口内不是最大，该像素置0 
5. 选择非零点作为角点检测结果

**非极大值抑制**

```bash
function Re = notMax(I, m, n, size)
% NOTMAX Harris角点非极大值抑制
% I: 输入Harris响应矩阵, m:行数, n:列数, size:邻域窗口边长
half = floor(size / 2);
Re = zeros(m, n); % 初始化结果矩阵
for i = 1:m
    for j = 1:n
        % 确定邻域上下左右边界，防止越界
        row_min = max(1, i - half);
        row_max = min(m, i + half);
        col_min = max(1, j - half);
        col_max = min(n, j + half);
        % 截取局部邻域
        local_patch = I(row_min:row_max, col_min:col_max);
        max_val = max(local_patch(:)); % 邻域最大值
        if I(i,j) == max_val
            Re(i,j) = I(i,j); % 局部极大值保留
        else
            Re(i,j) = 0;      % 非极大值抑制置0
        end
    end
end
end
```

**SIFT特征检测的步骤**

1. 检测尺度空间的极值点
2. 精确定位特征点(Keypoint)
3. 设定特征点的方向参数
4. 生成特征点的描述子（128维向量）

SIFT首先构建高斯尺度空间和DoG金字塔；

然后在DoG尺度空间中寻找局部极值点，每个点与同层和相邻尺度的邻域点比较；

接着通过插值精确定位关键点，并去除低对比度点和边缘响应点；

之后根据关键点邻域梯度方向直方图分配主方向，使特征具有旋转不变性；

最后在关键点邻域中划分 4×4 个子区域，每个区域统计8个方向梯度直方图，形成 4×4×8=128 维描述子，并进行归一化以增强对光照变化的鲁棒性。

**匹配**

对两幅图分别提取SIFT关键点和128维描述子；

对每个描述子在另一幅图中用欧氏距离寻找最近邻和次近邻；

若最近邻距离与次近邻距离之比小于阈值，例如0.75，则接受该匹配，否则认为匹配不可靠；

最后可用RANSAC根据几何一致性进一步剔除误匹配。

**HOG的基本思想**

- 局部目标的外表和形状可以被局部梯度或边缘方向的分布很好的描述，即使不知道对应的梯度和边缘的位置
- 将图像分为细胞单元(cells)，计算细胞单元的梯度方向直方图
- 把细胞单元组合成块(block)并归一化，形成HOG描述子

**HOG的生成步骤**

1. 图像归一化
2. 计算图像梯度
3. 在细胞单元构建梯度直方图
4. 块内梯度直方图对比度归一化
5. 计算HOG特征描述子

**基于Hough变换的线检测**

- 基本原理：点－线的对偶性质

算法： 基于Hough变换的直线检测

1. 将参数空间细分为m×n个累加单元，并设置累加器矩阵Q
2. 将累加器矩阵 Q(m×n)的初始值置为零
3. 对于XY平面上的点(xi，yi)，令θ等于θ轴上所有允许的细分值，计算对应的𝜌
4. 将得到的𝜌值四舍五入为最接近𝜌轴上的允许细分值
5. 在累加器矩阵Q中，找到所对应的单元，并将该单元的累加值加1 
6. 重复3-5步直到处理完XY平面上的所有点，在累加器矩阵Q中，值最大单元对应的θ和𝜌即为XY平面直线方程的参数
